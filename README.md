# The Bowler

A **mobile-first 2D scroller** where Carol the Bowler rides a scooter uphill and tosses her pink bowling ball at marching enemies.

## Run locally

### Option A — Open directly
1. Open `index.html` in a browser.
2. For best results, use a mobile device or a responsive viewport in the browser.

### Option B — Simple local server (recommended)

If you have a local server tool (like `serve`), run:

```bash
cd the-bowler
npx serve .
```

Then open the served URL in a browser.

## Gameplay

- Tap to start.
- Pull back from Carol (drag on the player) to aim and release to throw the pink bowling ball.
- Hit marching enemies to score points.
- Each time you reach the top of the hill, the level ends.
- Your best score is saved locally.

## Development

This project is intentionally small and dependency-free.

### Suggested tooling
- A modern browser (Chrome, Safari, Edge)
- A code editor (VS Code, etc.)

### Improving the game
- Add sound effects (Web Audio API)
- Add more enemy types and behaviors
- Add level progression (multiple slopes, hazards)
- Add UI polish (menus, animations, better character art)
